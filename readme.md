zerodha clone is a full-stack web application built with the **MERN stack** (MongoDB, Express.js, React.js, Node.js). It allows users to track stocks, manage a virtual portfolio, and simulate buy/sell operations. The project focuses on clean UI, modular code structure, and real-world trading functionalities.

---

## **Features**

### Authentication
- User signup and login using JWT tokens  
- Secure password storage and validation  
- Role-based access control for future scalability  

### Dashboard
- Interactive dashboard showing portfolio overview  
- Real-time portfolio value updates (simulated)  
- Watchlist for favorite stocks  

### Trading
- Simulate buy and sell orders  
- Automatic portfolio updates based on trades  
- Transaction history for all actions  

### Portfolio & Analytics
- Track total portfolio value and individual stock performance  
- Calculate profit/loss per stock  
- Visual charts using Chart.js for stock trends  

### UI/UX Improvements
- Clean and responsive design for desktop and mobile  
- Custom theme with unique colors and fonts  
- Modular React components for easy maintenance
